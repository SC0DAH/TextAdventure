# TextAdventure

## Course
**Testing & Security**

---

## Project Overview
TextAdventure is a project developed as part of the *Testing & Security* course.  
The goal of this project was to apply both software testing principles and security concepts in a console game where the user needs to go through certain rooms.

---

## What We Learned

### Testing
- Writing **unit tests**
- Using **mocking** to isolate dependencies
- Writing scenarios in **Gherkin**

###  Security
- **Encrypting and decrypting** data & files
- Working with **JWT tokens** for authentication
- Understanding secure handling of sensitive information
